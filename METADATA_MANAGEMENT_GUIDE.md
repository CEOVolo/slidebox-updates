# Руководство по управлению метаданными

## Обзор системы метаданных

Система поддерживает следующие типы метаданных для слайдов:

### 1. Основные поля
- **Статус** - draft, in_review, approved, archived
- **Формат** - vertical, horizontal  
- **Solution Areas** - области решений (marketing, sales, engineering и т.д.)

### 2. Локализация
- **Язык** - en, fr, de, multilang
- **Регион** - global, emea, na, apac, latam

### 3. Бизнес-домены
15 доменов/индустрий:
- automotive, fintech, retail, healthcare, education
- ecommerce, manufacturing, consulting, public-sector
- government, defense, logistics, telecom
- travel, entertainment

### 4. Дополнительные
- **Имя автора** - текстовое поле
- **Case Study** - флаг да/нет
- **Годы проекта** - начало и окончание

## Где управлять метаданными

### В настройках (/settings)
- **Solution Areas** - полное управление
  - Добавление новых (код + название)
  - Редактирование названий
  - Удаление (с проверкой использования)
- **Теги** - полное управление
- **Просмотр** всех остальных полей

### В модерации (/moderation)
- **Редактирование одного слайда**
  - Кнопка "Edit Details" на карточке
  - Все поля доступны для изменения
  
- **Массовое редактирование**
  - Выберите слайды галочками
  - Нажмите "Edit Metadata"
  - Измените нужные поля для всех выбранных

- **Автозаполнение**
  - Кнопка "Auto Extract"
  - Автоматически определяет:
    - Название слайда
    - Домен по ключевым словам
    - Solution areas по содержимому  
    - Case study по заголовку
    - Генерирует теги

## Рекомендации

1. **Используйте Auto Extract** для новых слайдов - это сэкономит время
2. **Проверяйте автозаполнение** - иногда требуется ручная корректировка
3. **Используйте массовое редактирование** для однотипных изменений
4. **Создавайте Solution Areas** заранее для новых направлений бизнеса
5. **Следите за единообразием** названий и кодов

## Примеры использования

### Добавление новой Solution Area
1. Настройки → Метаданные
2. В поле "Код" введите: `analytics`
3. В поле "Название" введите: `Analytics`
4. Нажмите кнопку "+"

### Массовое изменение домена
1. Модерация → Выберите слайды галочками
2. Нажмите "Edit Metadata"
3. В поле "Domain" выберите нужный
4. Нажмите "Update X slides"

### Автоматическое заполнение
1. Модерация → Найдите слайд
2. Нажмите "Auto Extract"
3. Проверьте результат в модальном окне
4. При необходимости откорректируйте
